## Project Overview
This project focuses on analyzing data for a shipping company to understand
sales performance, profit behavior, discounts impact, and shipment-related
factors. The project also applies machine learning techniques to evaluate
business performance and support data-driven decisions.

## Objectives
- Analyze sales and profit trends
- Study the impact of discounts on company performance
- Prepare and clean data for analysis
- Apply machine learning models for prediction and evaluation

## Dataset
- The dataset is provided as an Excel file.
- It includes features such as sales, profit, discount, quantity, and shipping-related attributes.
- The dataset is stored inside the `data/` folder.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook (Google Colab)

## Project Workflow
1. Data Loading
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering & Encoding
5. Train-Test Split
6. Model Training (Random Forest)
7. Model Evaluation (Accuracy & Classification Metrics)

## Results & Insights
- The model performance was evaluated using accuracy and classification metrics.
- The analysis provides insights that can help improve decision-making
  within the shipping company.
